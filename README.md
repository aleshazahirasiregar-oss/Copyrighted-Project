## Build Process
1. Pastikan Python 3.10+ sudah terinstal.
2. Clone repository: `git clone [url-repo-jendral]`
3. Jalankan script builder: `python build_client.py`
4. Hasil .apkg akan tersimpan di direktori `/dist`.
